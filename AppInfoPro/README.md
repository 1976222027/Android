# [AppInfoPro](https://github.com/afkT/Android/tree/master/AppInfoPro)

Android - 获取手机安装的应用信息（用户应用、系统应用）、手机信息、屏幕信息，并且支持扫描本地符合指定要求（后缀）的应用，并且获取该 APK 信息，支持导出应用信息（JSON格式）、分享应用，导出应用等功能
